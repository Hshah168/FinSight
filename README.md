# FinSight

End-to-end Python-based ETL pipeline processing 1,200+ financial transactions, transforming raw GL and budget data into structured fact/dimension tables optimized for analytical queries and BI modeling. Automated executive KPI calculations (EBITDA, Gross Margin %, Operating Margin %, YOY variance) using python pandas & SQL, eliminating 20+ hours of monthly manual reporting work. Engineered automated variance flagging logic identifying performance deviations greater than 10-15% across 600+ account period combinations, enabling proactive financial decision making.

## Required Tools
* Python
* Visual Studio Code
* Power Bi
* GitHub 

## Tools and Technologies learnt/used in making this project
* Python for building entire KPI engine and variance system
* Financial storytelling in Power BI
* AI to debug
